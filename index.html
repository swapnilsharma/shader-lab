About Frakt
BETA v0.6

Frakt started as a selfish project. I wanted beautiful shader backgrounds for my own work but every path to getting them required either writing GLSL from scratch or paying for tools that locked the output behind their ecosystem.

So I built Frakt — a layer-based shader creator that exports real, runnable code. The name is a compression of "fragment" and "fractal".
